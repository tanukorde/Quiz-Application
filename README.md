# 🎯 Quiz Application

## 📌 Project Description
The Quiz Application is a simple and interactive web-based project that allows users to attempt multiple-choice questions (MCQs), test their knowledge, and view their score instantly at the end of the quiz.

---

## 🚀 Features
- Multiple Choice Questions (MCQ)
- Instant Result Display
- Score Calculation
- Restart Quiz Option
- Simple and Clean UI
- Responsive Design (Mobile + Desktop)

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

Quiz-Application/
│── index.html

---

## 🎮 How to Run This Project
1. Clone the repository using:
   git clone https://github.com/tanukorde/Quiz-Application.git

2. Open the project folder

3. Open `index.html` in your browser

4. Start the Quiz 🎉
